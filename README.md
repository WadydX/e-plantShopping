# e-plantShopping

A React-based e-commerce application for browsing and purchasing plants. This project was built as part of a skills network course to practice React, Redux, and Vite.

## Features

- Browse plants by category (Air Purifying, Aromatic, Insect Repellent, Medicinal, Low Maintenance)
- Add plants to a shopping cart
- View and manage cart items (update quantities, remove items)
- Calculate total cart amount

## Tech Stack

- React
- Redux Toolkit
- Vite

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev` to start the development server
